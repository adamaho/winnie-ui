## Import

Import just the listbox styles.

```css
@import "@winnie-ui/component/listbox";
```

## Anatomy

Attach the appropriate `data-component` attributes in the structure below.

```html
<ul data-component="listbox">
  <li data-component="listbox-item">
    <span data-component="label"></span>
  </li>
  <div data-component="separator"></div>
  <section data-component="section">
    <header data-component="header"></header>
    <li data-component="listbox-item">
      <i data-component="indicator" class="check"></i>
      <span data-component="label"></span>
    </li>
    <li data-component="listbox-item">
      <i data-component="icon" class="settings"></i>
      <span data-component="label"></span>
    </li>
  </section>
</ul>
```
