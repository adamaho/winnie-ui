## Import

Import all of the field styles.

```css
@import "@winnie-ui/component/description";
@import "@winnie-ui/component/error";
@import "@winnie-ui/component/field";
@import "@winnie-ui/component/label";
```

## Anatomy

Attach the appropriate `data-component` attributes in the structure below.

```html
<div data-component="field">
  <label data-component="label" />
  <span data-component="description" />
  <!-- control component -->
  <span data-component="error" />
</div>
```
