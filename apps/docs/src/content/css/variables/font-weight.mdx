---
title: Font Weight
description: Variables for modifying the font weight of an element
---

## Import

Import just the typography variables.

```css
@import "@winnie-ui/base/typography";
```

## Reference

Below is a list of the available font weight variables in Winnie CSS.

| Variable                       | Tailwind Class    |
| ------------------------------ | ----------------- |
| `--wui-font-weight-normal`     | `font-normal`     |
| `--wui-font-weight-medium`     | `font-medium`     |
| `--wui-font-weight-semibold`   | `font-semibold`   |
| `--wui-font-weight-bold`       | `font-bold`       |
| `--wui-font-weight-extra-bold` | `font-extra-bold` |
