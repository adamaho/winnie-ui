---
title: Border Width
description: Variables for changing the border width of an element
---

## Import

Import just the border variables.

```css
@import "@winnie-ui/base/border";
```

## Reference

Below is a list of the available border width variables in Winnie CSS.

| Variable                | Tailwind Class |
| ----------------------- | -------------- |
| `--wui-border-width-05` | `border-1`     |
| `--wui-border-width-1`  | `border-2`     |
| `--wui-border-width-2`  | `border-3`     |
| `--wui-border-width-3`  | `border-4`     |
