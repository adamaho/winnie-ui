---
title: Font Family
description: Variables for modifying the font family of an element
---

## Import

Import just the typography variables.

```css
@import "@winnie-ui/base/typography";
```

## Reference

Below is a list of the available font family variables in Winnie CSS.

| Variable                  | Tailwind Class |
| ------------------------- | -------------- |
| `--wui-font-family-sans`  | `font-sans`    |
| `--wui-font-family-mono`  | `font-mono`    |
| `--wui-font-family-serif` | `font-serif`   |
