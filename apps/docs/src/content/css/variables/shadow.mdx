---
title: Shadow
description: Variables for modifying the box shadow of an element
---

## Import

Import just the shadow variables.

```css
@import "@winnie-ui/base/shadow";
```

## Reference

Below is a list of the available shadow in Winnie CSS.

| Variable               | Tailwind Class   |
| ---------------------- | ---------------- |
| `--wui-shadow-1`       | `shadow-1`       |
| `--wui-shadow-2`       | `shadow-2`       |
| `--wui-shadow-3`       | `shadow-3`       |
| `--wui-shadow-4`       | `shadow-4`       |
| `--wui-shadow-5`       | `shadow-5`       |
| `--wui-shadow-6`       | `shadow-6`       |
| `--wui-shadow-inner-1` | `shadow-inner-1` |
| `--wui-shadow-inner-2` | `shadow-inner-2` |
| `--wui-shadow-inner-3` | `shadow-inner-2` |
| `--wui-shadow-inner-4` | `shadow-inner-4` |
| `--wui-shadow-inner-5` | `shadow-inner-5` |
