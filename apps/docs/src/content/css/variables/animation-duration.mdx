---
title: Animation Duration
description: Variables for changing animation and transition duration
---

## Import

Import just the animation variables.

```css
@import "@winnie-ui/base/animation";
```

## Reference

Below is a list of the available animation duration variables in Winnie CSS.

| Variable            | Tailwind Class |
| ------------------- | -------------- |
| `--wui-duration-05` | `duration-05`  |
| `--wui-duration-1`  | `duration-1`   |
| `--wui-duration-2`  | `duration-2`   |
| `--wui-duration-3`  | `duration-3`   |
| `--wui-duration-4`  | `duration-4`   |
