---
title: Animation Timing Function
description: Variables for changing the timing of an animation or transition
---

## Import

Import just the animation variables.

```css
@import "@winnie-ui/base/animation";
```

## Reference

Below is a list of the available animation timing variables in Winnie CSS.

| Variable         | Tailwind Class  |
| ---------------- | --------------- |
| `--wui-bezier-1` | `ease-bezier-1` |
