---
title: Font Size
description: Variables for modifying the font size of an element
---

## Import

Import just the typography variables.

```css
@import "@winnie-ui/base/typography";
```

## Reference

Below is a list of the available font size variables in Winnie CSS.

| Variable            | Tailwind Class |
| ------------------- | -------------- |
| `--wui-font-size-1` | `text-1`       |
| `--wui-font-size-2` | `text-2`       |
| `--wui-font-size-3` | `text-3`       |
| `--wui-font-size-4` | `text-4`       |
| `--wui-font-size-5` | `text-5`       |
| `--wui-font-size-6` | `text-6`       |
| `--wui-font-size-7` | `text-7`       |
| `--wui-font-size-8` | `text-8`       |
| `--wui-font-size-9` | `text-9`       |
